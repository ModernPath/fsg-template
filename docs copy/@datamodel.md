### `financial_metrics`
Stores calculated or extracted financial metrics for a company for a specific fiscal period.

-   `id` (UUID, PK): Unique identifier for the metric record.
-   `company_id` (UUID, FK -> companies): The company these metrics belong to.
-   `fiscal_year` (INTEGER): The fiscal year the metrics apply to.
-   `fiscal_period` (TEXT): The period within the year (e.g., 'annual', 'q1', 'h1').
-   `revenue_current` (DECIMAL): Revenue for the current period.
-   `revenue_previous` (DECIMAL): Revenue for the previous period.
-   `revenue_growth_rate` (DECIMAL): Calculated revenue growth rate (%).
-   `operating_profit` (DECIMAL): Operating Profit / EBIT.
-   `net_profit` (DECIMAL): Net Profit / Net Income.
-   `depreciation_amortization` (DECIMAL): Depreciation and Amortization expense.
-   `operational_cash_flow` (DECIMAL): Cash flow from operating activities.
-   `investment_cash_flow` (DECIMAL): Cash flow from investing activities.
-   `financing_cash_flow` (DECIMAL): Cash flow from financing activities.
-   `return_on_equity` (DECIMAL): Return on Equity (ROE) percentage.
-   `debt_to_equity_ratio` (DECIMAL): Debt-to-Equity (D/E) ratio.
-   `quick_ratio` (DECIMAL): Quick ratio (acid-test ratio).
-   `current_ratio` (DECIMAL): Current ratio.
-   `asset_structure_percent` (DECIMAL): Percentage of fixed assets to total assets.
-   `total_fixed_assets` (DECIMAL): Total value of fixed assets.
-   `accounts_receivable` (DECIMAL): Total accounts receivable.
-   `inventory` (DECIMAL): Total inventory value.
-   `current_assets` (DECIMAL): Total current assets.
-   `total_equity` (DECIMAL): Total equity.
-   `total_liabilities` (DECIMAL): Total liabilities.
-   `current_liabilities` (DECIMAL): Total current liabilities.
-   `accounts_payable` (DECIMAL): Total accounts payable.
-   `accounts_receivable_turnover_days` (INTEGER): Accounts receivable turnover in days.
-   `bad_debt_ratio` (DECIMAL): Bad debt ratio (%).
-   `fixed_asset_turnover` (DECIMAL): Fixed asset turnover ratio.
-   `investment_allocation` (JSONB): Structure detailing investment allocation (e.g., short vs. long term).
-   `data_source` (TEXT): Source of the data (e.g., 'document_extraction', 'company_metadata', 'public_financial_data').
-   `status` (TEXT): Status of the data (e.g., 'draft', 'complete', 'verified').
-   `financial_statements` (JSONB): Raw financial statement data if available.
-   `key_metrics` (JSONB): Raw key metrics extracted (e.g., from Gemini).
-   `source_document_ids` (UUID[]): Array of document IDs used as sources for these metrics.
-   `funding_recommendations_id` (UUID, FK -> funding_recommendations): Link to related funding recommendations.
-   `created_by` (UUID, FK -> auth.users): User who created/triggered the record.
-   `created_at` (TIMESTAMPZ): Timestamp of creation.
-   `updated_at` (TIMESTAMPZ): Timestamp of last update.
